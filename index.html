NP_CKEditor

CKEditor Plugin for Nucleus CMS

A plugin for Nucleus CMS that utilize CKEditor in Blog editing. Based on yama's original plugin on http://japan.nucleuscms.org/forum/viewtopic.php?id=4338, added support for CKEditor 4.5 by osamuh.

This project is maintained by NucleusCMS
